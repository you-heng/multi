#### 接口平台

dapinke -- 获取大拼客榜单
dataoke -- 获取大淘客榜单 或者 获取淘宝联盟物料精选
duoduo  -- 搜索多多进宝
taobao  -- 搜索淘宝联盟

## 使用说明

当前系统是一个淘宝拼多多领券的系统

开发者只需要在`api/Config.php`中将自己的key等信息

`api/clocking.php`是定时任务，定时刷新淘宝拼多多最新的领券连接，自己在服务器添加定时任务
