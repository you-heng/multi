## 目录

主要接口文件在`push`目录下

push

​	Config.php --------- 配置各大厂商后台的key等信息

​	database.php ----- 配置数据连接账号密码等

​	index.php ---------- 接口的入口文件

sql

​	push.sql

index.php ----------------- 测试发送页面



## 使用过程

1. 先配置`push/Config.php`中的key等信息
2. 将数据表文件添加到自己的数据中
3. 在`index.php`中根据各大厂商的文档填写对应的内容点击发送即可
4. 在数据库中查看发送状态是否成功
